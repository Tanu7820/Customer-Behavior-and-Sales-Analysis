# 📊 Customer Behavior & Sales Analysis (End-to-End Project)

## 📌 Project Overview
This project focuses on analyzing customer behavior and sales performance using an E-commerce/Retail transactional dataset. The objective is to convert raw data into meaningful business insights through structured data cleaning, transformation, and visualization.

The project follows a complete end-to-end analytics workflow, including data preprocessing in Python, exploratory analysis in Excel, KPI development using DAX, and interactive dashboard creation in Power BI. The final outcome helps in understanding customer purchasing patterns, identifying revenue drivers, and supporting data-driven business decisions.

The project uses **Excel, Python, and Power BI with DAX** to complete the end-to-end data analytics workflow.
